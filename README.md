# DesignPattern (MVC MVP MVVM) EX projects

## Usage
1. Move to target dir
```bash
cd ${project_dir}
ex) cd ./MVP
```
2. Configure Project
```bash
cmake -B build -DCMAKE_BUILD_TYPE=Release
```
3. Build Project
```bash
cmake --build build --config Release
```
4. Execute Project
```bash
./build/${project_name}
ex) ./build/mvc_project
```
